# 수식 계산기 (Formula Calculator)

사용자가 직접 변수와 수식을 정의하고, 여러 상품에 대해 일괄 계산 및 계산 기록(차트)을 관리할 수 있는 커스텀 수식 계산기 PWA(Progressive Web App)입니다. 모바일 기기(특히 갤럭시 S22 등)에 최적화된 UI를 제공합니다.

## ✨ 주요 기능

*   **⚙️ 기본설정**: 
    *   글씨 크기 조절
    *   자유로운 변수 추가/삭제 (상수, 변수, 종속변수로 타입 지정 가능)
    *   변수에 사용자 정의 이름(별칭) 부여 가능
    *   종속변수를 계산하기 위한 커스텀 수식 정의 (예: `A * B + 3`)
*   **📋 기본정보**: 
    *   계산 대상(상품명, 규격) 관리
    *   각 대상별 고정된 '상수' 값 입력
*   **🧮 계산**: 
    *   '변수' 값 입력 시 실시간으로 수식 자동 계산
    *   각 대상별(행별) 종속변수에 적용할 수식 개별 선택 가능
    *   계산 결과를 차트 데이터로 저장
*   **📊 차트**: 
    *   상품명과 규격 기준으로 계산 이력(입력값 및 결과값) 저장 및 조회
    *   전체 기록 초기화 기능
*   **📱 PWA 지원**: 모바일 웹 브라우저에서 '홈 화면에 추가'하여 네이티브 앱처럼 사용 가능
*   **💾 로컬 저장소**: 모든 설정과 데이터는 기기의 로컬 스토리지에 저장되어 앱을 종료해도 유지됨

## 🚀 설치 및 실행 방법

이 프로젝트는 서버 사이드 코드가 필요 없는 순수 프론트엔드 웹 애플리케이션입니다.

### 방법 1: 로컬에서 실행
1.  이 저장소를 클론하거나 ZIP 파일로 다운로드합니다.
2.  `index.html` 파일을 최신 웹 브라우저(Chrome, Safari 등)에서 엽니다.

### 방법 2: 웹 호스팅 (추천)
모바일 기기에서 PWA로 설치하여 사용하려면 웹 호스팅이 필요합니다.
1.  GitHub Pages, Vercel, Netlify 등의 정적 웹 호스팅 서비스에 이 저장소를 배포합니다.
2.  모바일 브라우저로 배포된 URL에 접속합니다.
3.  브라우저 메뉴에서 **'홈 화면에 추가'** 또는 **'앱 설치'**를 선택합니다.

## 🛠 기술 스택
